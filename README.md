### Desctiption

A mobile weather app displays the weather of a chosen location.

- now 
- today's maximum and minimum temperature
- every 3 hours
- one week from today

### Technologies
- HTML, CSS
- Expo 
- Javascript
- React Native
- React


### Setup

To run this project, download weather_app from github. 

```
$ cd ../weather_app
$ expo start
```
